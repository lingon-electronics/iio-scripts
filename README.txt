
This repository contains scripts to help and guide Linux users
to get started with the following IIO sensor products:

* 3D Accelerometer ("accel_3d")
* 3D Gyroscope ("gyro_3d")

For more information about how IIO sensor readout works,
see http://www.lingonelectronics.com


If you're a Windows 10 user who wants to learn more on how to interact
with sensors, please check the following repository:

https://github.com/Microsoft/Windows-universal-samples
